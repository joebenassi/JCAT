STARTUP INSTRUCTIONS (organized by JRE)
Note: The JRE must be Java 6 or later.

Windows 32-bit JRE:
1. Download "JCAT_win32.jar".
2. Download all App Profiles and Constant Definition files.
3. Double-click "JCAT_win32.jar".

Windows 64-bit JRE:
1. Download "JCAT_win64.jar".
2. Download all desired App Profiles and Constant Definition files.
3. Double-click "JCAT_win64.jar".

Linux 32-bit JRE:
1. Download "JCAT_linux32.jar".
2. Download all App Profiles and Constant Definition files.
3. Double-click "JCAT_linux32.jar".

Linux 64-bit JRE:
1. Download "JCAT_linux64.jar".
2. Download all desired App Profiles and Constant Definition files.
3. Double-click "JCAT_linux64.jar".

Macintosh OSX 32-bit JRE:
1. Download "JCAT_osx32.jar".
2. Download all App Profiles and Constant Definition files.
3. Open "Terminal".
4. Navigate to the folder containing "JCAT_osx32.jar".
5. Type in Terminal "java -XstartOnFirstThread -jar JCAT_osx32.jar".

Macintosh OSX 64-bit JRE:
1. Download "JCAT_osx64.jar".
2. Download all App Profiles and Constant Definition files.
3. Open "Terminal".
4. Navigate to the folder containing "JCAT_osx64.jar".
5. Type in Terminal "java -XstartOnFirstThread -jar JCAT_osx64.jar".
